# codeplay
# CODEPLAY Hackathon Website

Official website for the CODEPLAY Hackathon.

Live site: https://codeplay.hr

---

## About

This repository contains the source code for the CODEPLAY hackathon website, including:

- Main landing page
- Countdown subpage
- Event information
- Downloadable task materials
- Final results display (after the event)

The site is built using plain HTML, CSS, and JavaScript.

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Inter)

No external frameworks are used.

## Running Locally

1. Clone the repository:
git clone https://github.com/your-username/codeplay-website.git

2. Open `index.html` in your browser  
or use a local development server (recommended).
